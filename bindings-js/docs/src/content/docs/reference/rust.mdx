Install the `pluot` Rust crate from [crates.io](https://crates.io/crates/pluot).

```sh frame="none"
cargo add pluot
```

<Aside type="caution" title="Not yet published to Crates.io">
   For now, the crate is only available from GitHub; we have only reserved the crate name on Crates.io.
</Aside>
