Install the `pluot` Rust crate from [crates.io](https://crates.io/crates/pluot).

```sh frame="none"
cargo add pluot
```

<Aside type="caution" title="In flux">
    The Rust internals are still being actively developed and subject to frequent changes.
</Aside>

<LinkButton href="https://docs.rs/pluot/latest/pluot/">View on docs.rs</LinkButton>

## Examples

In the `examples` directory of the repository, you will find:


- a CLI that renders plots to SVG or PNG files, given a JSON file with plotting parameters
- an HTTP server that responds with an SVG string or PNG bytes, given a POST request with plotting parameters
- a GUI built with the [Egui](https://github.com/emilk/egui) framework
- a GUI built with the [Slint](https://github.com/slint-ui/slint) framework


<LinkButton href="https://github.com/keller-mark/pluot/tree/main/examples">View examples</LinkButton>
