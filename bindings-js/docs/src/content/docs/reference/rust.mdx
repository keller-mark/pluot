Install the `pluot` Rust crate from [crates.io](https://crates.io/crates/pluot).

```sh frame="none"
cargo add pluot
```

<Aside type="caution" title="In flux">
    The Rust internals are still being actively developed and subject to frequent changes.
</Aside>
