## Per-point (instanced) size and color

`point_radius`, `fill_color`, and the stroke properties all accept instanced modes, supplying one value per point.

<PluotWrapper
    client:load
    client:only="react"

    plotParams={{
      layers: [
        {
          layer_type: "PointLayer",
          layer_params: {
            layer_id: "instanced_points",

            data_unit_mode_y: "Data",
            point_shape_mode: "Circle",
            point_radius_unit_mode_x: "Data",

              size_mode: "InstancedSize",
